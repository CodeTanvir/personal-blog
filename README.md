# Personal Blog

A modern personal blog website featuring form validation, data storage, theme customization, and asynchronous data handling.

## Features

- User registration and login with validation
- Dark mode toggle with localStorage persistence
- Font size controls
- Asynchronous data loading for posts and comments
- Responsive design

## Project Structure

```
personal-blog/
├── src/
│   ├── html/
│   ├── css/
│   ├── js/
│   └── images/
├── dist/
├── package.json
└── README.md
```

## Development

1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Build for production: `npm run build`

## Deployment

Deployed on GitHub Pages/Netlify/Vercel.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Parcel (build tool)
- localStorage
- Fetch API
